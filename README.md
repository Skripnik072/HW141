# Проект PythonPHome2

## Описание:

Проект это веб-приложение на Python для работы с классами

## Установка:

1. Клонируйте репозиторий:
```
git clone git@github.com:Skripnik072/HW111.git
```

## Использование:

1. Приложение создает два класса и считает количество категорий продуктов.
2. Приложение создает дочерние классы, наследованные от класса Product.
3. Позволяет складывать экземляры одного класса и не позволяет складывать разные.
4. Не позволяет добавлять экземляры, наследованные от других классов.
