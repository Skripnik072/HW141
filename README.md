# Проект PythonPHome2

## Описание:

Проект это веб-приложение на Python для работы с классами

## Установка:

1. Клонируйте репозиторий:
```
git clone git@github.com:Skripnik072/HW141.git
```

## Использование:

1. Приложение создает классы и считает количество категорий продуктов.
2. В приложение добавлен абстрактный класс.
3. В абстрактном классе добавлена общая функциональность для дочерних классов
4. Добавлена обработка исключений
