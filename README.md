# Проект PythonPHome2

## Описание:

Проект это веб-приложение на Python для работы с банковскими операциями

## Установка:

1. Клонируйте репозиторий:
```
git clone git@github.com:Skripnik072/HW102.git
```

## Использование:

1. Модуль select_dict предназначен для фильтрации словаря транзакций по подстроке из описания транзакции.
2. Модуль transf_list преобразует список словарей транзакций в словарь, где ключи - названия категорий
    транзакций, а значения - количество операций в данной категории
3. Модуль main формирует словри транзакций из JSON, CSV и XLSX файлов, а также позволяет отфильтровать полученные
    словари по различным условиям.
